# US 3.2 - Export MP4

**Epic:** 3 - Génération & Export (Output)
**Status:** Pending
**Priority:** High

## Description
L'application génère un fichier vidéo MP4 téléchargeable (1080x1920 pour Reels) via l'enregistrement client-side.

## Acceptance Criteria
- [ ] Bouton "Exporter" ou "Générer Reel".
- [ ] La vidéo est générée au format MP4.
- [ ] La résolution est de 1080x1920 (format vertical 9:16).
- [ ] Le fichier est téléchargé automatiquement à la fin du processus.
