# US 1.2 - Prévisualisation 3D

**Epic:** 1 - Import & Visualisation (Core)
**Status:** Pending
**Priority:** High

## Description
L'utilisateur doit pouvoir manipuler le mug (rotation orbitale) pour vérifier le placement du design.

## Acceptance Criteria
- [ ] L'utilisateur peut faire tourner le mug à 360 degrés (orbite).
- [ ] Le zoom (in/out) est possible (optionnel mais recommandé).
- [ ] La rotation est fluide et réactive.
