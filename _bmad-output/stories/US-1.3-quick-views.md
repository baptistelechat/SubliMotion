# US 1.3 - Vues Rapides

**Epic:** 1 - Import & Visualisation (Core)
**Status:** Pending
**Priority:** Medium

## Description
Des boutons doivent permettre de s'aligner rapidement sur les faces clés (Face, Anse, Haut, Bas).

## Acceptance Criteria
- [ ] Présence de boutons d'accès rapide (Front, Back, Left, Right, Top, Bottom ou similaires).
- [ ] Au clic, la caméra s'anime pour se positionner sur la vue demandée.
